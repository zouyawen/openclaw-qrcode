# Contributing to OpenClaw QR Code Plugin

Thank you for your interest in contributing! This document outlines the guidelines for contributing to this plugin.

## Development Setup

1. Clone the repository
2. Install dependencies: `npm install`
3. Test locally by copying to `~/.openclaw/plugins/`

## Testing

- Test all features: generation, decoding, beautification
- Test with different channels (WebChat, Telegram, WhatsApp)
- Test error cases and edge cases

## Pull Requests

- Include clear description of changes
- Update documentation if needed
- Ensure all tests pass

## Code Style

- Follow existing code patterns
- Add comments for complex logic
- Handle errors gracefully

## Reporting Issues

- Include OpenClaw version
- Provide reproduction steps
- Attach relevant logs or screenshots
